Settings

Profile

Displays your username and email address. These are set during account creation and cannot be changed from the UI.

Change Password

Enter your current password and a new password (minimum 8 characters). Click "Save Password" to update. You'll stay logged in after the change.

Invite Users (Admin Only)

Admins can invite new users by entering their email and desired username. An invitation email is sent (if SMTP is configured) containing a link the recipient uses to set their password. Existing invites are listed with "Revoke" buttons to cancel pending invitations.

Brave Search API Key

Used for web search in Deep Research. Enter your Brave Search API key (format: BSA-...) and click "Save Key". Once saved, the key is displayed in masked form (BSA-...xyz). Click "Remove Key" to delete it.

Tip: Brave Search offers a free tier with 2,000 queries/month. Sign up at brave.com/search/api.

Email Configuration (Admin Only)

Configure SMTP settings for sending password reset emails and user invitations. Fields: SMTP host, port (default 587), username, password, from address. A Google API token textarea is available for Gmail OAuth2 authentication.

Inference Providers

Manage your LLM API connections. Each provider has:

  • Name — display label (e.g. "OpenRouter", "OpenAI")
  • API Key — the provider's API key (displayed masked after saving)
  • URL — the API endpoint (default: https://openrouter.ai/api/v1)
  • Strong Model — model ID for complex tasks (e.g. deepseek/deepseek-v4-pro)
  • Quick Model — model ID for fast/lightweight tasks (e.g. deepseek/deepseek-v4-flash)
  • Rate Limit — requests per minute (0 = unlimited)
  • Set Default — makes this provider the default for all agents

Click "Add Provider" to create a new one. Each provider card has Edit and Delete buttons.

Theme

Toggle between Dark and Light mode. The button label updates to reflect the current state ("Switch to Light Theme" vs "Switch to Dark Theme"). Your preference is saved to localStorage and persists across sessions.

Agent Settings

A grid of cards for each registered agent. Each card shows the agent name, description, and version. Features:

  • Enable/Disable toggle — hides the agent tab from the header when disabled
  • Plugin settings form — dynamically generated form fields based on the agent's schema. Field types include text, number, boolean (checkbox), enum (select), and others
  • Save Settings — persists agent-specific configuration

My API Keys

Create and manage personal API keys for programmatic access. Keys use the wb- prefix. Each key requires a descriptive name (label). When created, the full key is shown once — copy it immediately as it cannot be retrieved later. Keys can be deleted individually.

Limit: Maximum 5 API keys per user.
