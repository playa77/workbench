Session History

Overview

Session History provides a chronological view of all your agent sessions — debates, research runs, planning sessions, chat conversations, and more. Browse, search, view details, and export results.

Agent Filter

Filter the session list by agent type. Select "All" to see all sessions or choose a specific agent (Chat, Debate, Research, Planning, Consigliere, News, Math Tutor, Knowledge Base) to narrow results.

Session Table

Each row shows: agent type, session timestamp, status indicator, and action buttons:

  • View — loads the full session detail (transcript, report, plan, etc.)
  • PDF — exports the session as a PDF document
  • Delete — permanently removes the session. Irreversible.

Session Detail View

Clicking "View" opens the session detail. Content depends on the agent type — debates show transcripts, research shows the report, plans show the structured plan. Use "Back to List" to return to the table.

PDF Export

Click the "PDF" button on any session row to export. The button shows "Generating PDF..." during export. Choose a template using the dropdown in the filter row.

PDF Style Selector

Choose a PDF template from the dropdown labeled "PDF Style:" in the filter bar above the session table.

Tip: Use the browser's back/forward buttons to navigate between the session list and detail views — history navigation is fully supported.
