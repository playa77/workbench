News Pipeline

Overview

The News Pipeline aggregates content from RSS/Atom feeds, processes articles through your LLM, and produces summaries and analysis. Create "interests" — topic-focused feed collections — and run them on demand or on a schedule.

Interests

An interest is a named collection of feeds with processing settings. Each interest card displays its name, feed count, schedule interval, and action buttons.

Creating an Interest

Use the "Add Interest" form at the top of the page:

  • Name — descriptive label for this topic (e.g. "AI Research", "Climate Science")
  • Start Date — only process articles published after this date
  • Interval (hours) — how often the pipeline runs automatically
  • Summary Prompt — custom LLM prompt for summarizing articles
  • Script Prompt — custom LLM prompt for generating scripts/analysis
  • Toggles — enable/disable summarizing, script generation, or scheduled auto-runs

Click "Create Interest" to save.

Editing an Interest

Click the "Edit" button on any interest card to open a pre-filled form with all current settings. Modify any field and save to update.

Managing Feeds

Click the "Feeds" button to open the feed management panel for that interest. Add new feeds by entering a name and RSS/Atom URL. Remove feeds with the delete button next to each entry.

Running a Pipeline

Click "Run Now" to execute the pipeline immediately. The button shows a loading state while the pipeline fetches, processes, and summarizes articles. Click "Load Runs" to view past pipeline execution results.

Deleting an Interest

Click "Delete" to remove the interest and all associated data. This action cannot be undone.

Scheduling

If the "Auto Run" toggle is enabled for an interest, the pipeline runs automatically at the specified interval. The background scheduler manages execution across all users' interests.

Tip: Start with a small interval (1-6 hours) and a narrow date range to test pipeline quality before scaling up.
