Open WebUI Integration

Overview

Open WebUI is an optional integrated chat interface that runs alongside Workbench. When enabled, it provides an alternative chat experience with model management, conversation history, and a modern chat UI. Access it as an embedded iframe within the Workbench app or open it in a separate tab.

Health Check

The status display shows whether Open WebUI is reachable:

  • Green dot — Active: Open WebUI is running and healthy
  • Gray dot — Unreachable: Open WebUI is not responding
  • Red dot — Error: Open WebUI returned an error

Retry Button

If Open WebUI is unreachable, click "Retry" to re-check the connection. This performs a fresh health check against the Open WebUI backend.

Open in New Tab

Click this link to open Open WebUI in a separate browser tab at /open-webui/. This gives you the full Open WebUI interface without the Workbench iframe wrapper.

Embedded View

When Open WebUI is healthy, it appears as an embedded iframe within the Workbench tab. All Open WebUI features (chat, model management, document upload) are available within this embed.

Note: Open WebUI runs as a separate Docker container. If it's not healthy, verify the container is running: docker compose ps open-webui
