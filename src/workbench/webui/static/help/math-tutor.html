Math Tutor

Overview

The Math Tutor agent provides step-by-step math tutoring with LaTeX equation rendering. It supports free-form problem entry, a visual equation builder, live chat Q&A during sessions, and comprehensive multiple-choice interviews for assessment.

Problem Text

Enter a math problem in natural language or LaTeX notation. Examples: "Solve the quadratic equation x^2 - 5x + 6 = 0", "Find the derivative of sin(x^2)", "Prove that sqrt(2) is irrational"

Equation Builder

Click "Equation Builder" to reveal a visual tool for constructing equations. Select equation type (polynomial, trigonometric, exponential, etc.), order, and coefficients. Click "Build Equation" to insert the generated LaTeX into the problem field.

Start Tutor Session

Click to begin a tutoring session. The agent initializes with your problem and begins step-by-step guidance. Progress streams via SSE.

Chat During Session

Use the chat input and Send button to ask follow-up questions during an active session. The tutor responds with LaTeX-rendered explanations.

Build Equation (in Chat)

During a session, click "Build Equation" to construct an equation to send in your message — useful for asking about specific mathematical expressions.

Comprehensive MC Interview

Click to launch a structured multiple-choice assessment. The agent generates conceptual questions about the current topic. Answer by clicking the radio options and click Submit Answer. Navigation buttons let you move Next/Back/Skip/Finish through questions.

End Session

Click to terminate the tutoring session. The full transcript is saved to your session history.

Tip: Use the Equation Builder when you need precise mathematical notation — the LaTeX output renders beautifully in the chat.
