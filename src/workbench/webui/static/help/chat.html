Chat Agent

Overview

The Chat agent provides a direct conversational interface with your configured LLM. It's the simplest way to interact with an AI model — type a message, get a response.

Model Selector

Choose which model to use for the conversation. The dropdown lists models from your configured inference providers. The default model from your default provider is pre-selected.

Message Input

Type your message in the text field at the bottom of the chat area. Press Enter to send. The chat scrolls automatically to show the latest messages.

Send Button

Click to submit your message. The button shows a loading state while the LLM generates a response. Messages appear chronologically with user messages on the right and AI responses on the left.

Past Chat Sessions

Click the toggle to expand the list of previous chat sessions. Each entry shows a preview of the conversation. Click a session to load its full message history. Sessions are preserved across tab switches and browser restarts.

Tip: The chat agent uses the quick model from your default inference provider. Configure this in Settings → Inference Providers.
