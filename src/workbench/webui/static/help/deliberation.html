Consigliere / Deliberation

Overview

The Consigliere agent stress-tests ideas through structured deliberation. It examines your proposal through multiple analytical frames, performs critique rounds, and optionally runs rhetoric analysis and synthesis.

Question / Proposal

Enter the idea, proposal, or decision you want to analyze. Be thorough — include context, constraints, and any assumptions. Example: "We should migrate our monolith to microservices because..."

Analytical Frames

Select one or more analytical perspectives. Each frame examines your proposal from a different angle:

  • Risk Analysis — identifies potential failure modes and their severity
  • Cost-Benefit — weighs investment against expected returns
  • Ethical — examines moral implications and stakeholder impacts
  • Feasibility — assesses technical and organizational practicality
  • Strategic Fit — evaluates alignment with broader goals
  • Additional frames may be available

Critique Rounds

Number of refinement iterations (1–5). Each round deepens the analysis by challenging previous findings. More rounds produce more nuanced results. Default: 2.

Temperature

Controls the creativity/randomness of the analysis (0.0–2.0). Lower values produce more deterministic, focused analysis. Higher values introduce more creative angles and unexpected perspectives.

Rhetoric Analysis

When enabled, the agent analyzes the rhetorical structure of your proposal — identifying persuasive techniques, logical fallacies, and argumentation patterns.

Synthesis

When enabled, the agent produces a unified synthesis combining insights from all frames into a single coherent recommendation.

Stress-Test Idea

Click to begin deliberation. Progress streams via SSE with phase indicators showing which frame is currently active.

Export

  • Export JSON — downloads the full deliberation results as structured JSON
  • New Session — clears results and returns to the input form

Past Sessions

Expand to view previous deliberation sessions. Click any entry to reload its results.

Tip: Enable both Rhetoric Analysis and Synthesis for the most comprehensive output — the agent will critique how you argue AND what you propose.
