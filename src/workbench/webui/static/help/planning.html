Strategic Planning

Overview

The Strategic Planning agent generates structured plans across 9 distinct plan types. Each plan type uses a specialized prompt and generation strategy tailored to the planning domain.

Goal / Objective

Describe what you want to achieve. Be specific about scope, timeline, resources, and constraints. Example: "Launch a SaaS product for remote team collaboration by Q4 with a team of 3 engineers"

Plan Type

Choose from 9 specialized plan formats:

  • Project Plan — phases, milestones, deliverables, dependencies
  • Business Strategy — market analysis, competitive positioning, growth levers
  • Technical Architecture — system design, stack choices, scalability
  • Marketing Campaign — channels, messaging, timeline, metrics
  • Product Roadmap — features, priorities, release schedule
  • Risk Mitigation — threat identification, likelihood, countermeasures
  • Learning Path — curriculum, resources, milestones, assessments
  • Operational Workflow — processes, roles, automation points
  • Financial Model — revenue, costs, projections, break-even

Model Selector

Choose which LLM to use for plan generation. The strong model from your default provider is pre-selected — planning benefits from more capable models.

Temperature

Controls creativity (0.0–2.0). Lower values produce more structured, conventional plans. Higher values generate more innovative and unconventional approaches. Default: 0.7.

Generate Plan

Click to start plan generation. Progress streams via SSE. The "Stop" button aborts a running generation.

Export & Copy

  • Copy — copies the plan text to clipboard
  • Export HTML — downloads as a formatted HTML file
  • Export PDF — generates a PDF using the selected template
  • New Plan — clears results and returns to the input form

Template Selector

Choose a PDF template for export.

Past Plans

Expand to view previous planning sessions. Click to reload a plan.

Tip: For complex goals, run the same objective through multiple plan types — a technical architecture plan and a project plan together give both the "what" and the "how".
