Login & Account

Overview

Workbench supports multiple authentication methods to suit different workflows. You can sign in with a username/password pair or use a personal API key for programmatic access.

Password Login

Email or Username

Enter the email address or username associated with your account. Both are accepted — the system looks up whichever you provide.

Password

Enter your account password. If you've forgotten your password, use the "Forgot password?" link below the sign-in button.

Sign In Button

Click to authenticate with your credentials. The button shows a loading spinner while the request is in progress. On success, you'll be taken to the main Workbench interface.

API Key Login

Paste a Workbench API key (format: wb-...) into the API Key field and click "Sign In with API Key". API keys bypass password authentication — they are intended for programmatic access, CI/CD pipelines, and external integrations. You can create and manage API keys in Settings → API Keys.

  • API keys have the same permissions as your user account.
  • You can create up to 5 keys per user.
  • Keys can be revoked individually at any time.

Forgot Password

Click the "Forgot password?" link on the login screen. Enter your registered email address and click "Send Reset Link". If an SMTP server is configured, you'll receive an email with a password reset link. The link expires after a set period.

Note: Password reset requires an SMTP server to be configured in Settings → Email Configuration. If no SMTP is set up, admins must reset passwords manually.

First-Time Setup

When Workbench is freshly installed with no users in the database, the setup screen appears automatically. Fill in:

  • Username — your display name (alphanumeric, no spaces)
  • Email — used for password recovery and notifications
  • Password — minimum 8 characters
  • Confirm Password — must match the password field

The first created user is automatically granted admin privileges. Click "Create Account" to complete setup.

Accepting an Invite

If an admin sends you an invitation link, clicking it takes you to the invite acceptance page. Enter your desired password, confirm it, and submit. Your account is created with the username and email the admin specified.
